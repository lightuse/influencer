    getTopNouns

    • getTopNouns(req: Request, res: Response): Promise<void>

      インフルエンサーの投稿から名詞の出現回数ランキングを取得します。

      指定したインフルエンサーIDの全投稿を解析し、最も多く使われている名詞を出現回数順に返します。 返却する名詞数はクエリパラメータlimitで指定できます（デフォルト: 10、最大: 100）。

      Parameters

      • req: Request

        Expressのリクエストオブジェクト。ルートパラメータとしてinfluencerId、クエリパラメータとしてlimitを受け取ります。

      • res: Response

        Expressのレスポンスオブジェクト。名詞ランキングまたはエラーメッセージをJSONで返します。

      Returns Promise<void>
